#Biblioteca Digital

Processo de construção:

1. Definir as entidades;
2. Identificar o que é independente e o que não é;
3. Começar pela classe independente (Acervo) que existe sozinha e vai usar seus próprios parametros;
   - classe Acervo possui um atributo privado que é um array de objetos #livros;
   - seu construtor deve iniciar tal array vazio;
   - os métodos vão equivaler às ações que a biblioteca faz (não necessariamente os botões).
   - Quais ações ela faz? Adiocionar, emprestar, devolver. Esses serão os métodos;
